📖 Project Development Stages

Stage 1: Architecture & UI Design

We started by defining the Cyber-Theme identity. Using Glassmorphism and CSS Blur effects to create a professional security interface.

Stage 2: Core Logic Implementation

Developing the RSA-simulation using JavaScript. We focused on handling Base64 encoding/decoding to represent secure data transmission.

Stage 3: Cloud Assets Integration

To ensure 100% stability, we migrated local image assets to Raw GitHub URLs, ensuring the background loads instantly on any server.

Stage 4: Interactive Presentation Layer

Adding the Single Page Application (SPA) navigation and timed result reveals to make the project suitable for live demonstrations.
