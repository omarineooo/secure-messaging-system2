The Project

All core information, origin, implementation flow, and security model of the system.

Project Information

  • Project Name: Secure Messaging System
  • Course: Security Two
  • Encryption Method: RSA
  • Password Protection: bcrypt
  • Backend: Node.js + Express
  • Database: SQLite

Project Origin

This project was created to apply cybersecurity concepts in a practical web application. It provides secure registration, authentication, encrypted messaging, user privacy, and controlled communication between registered users.

How It Works

1. Register

Create a private account with a unique username and protected password.

2. Login

Access the system securely from your device using your account credentials.

3. Send Message

Choose any registered user and send an RSA encrypted message securely.

4. Receive & Read

The receiver opens the inbox and reads the decrypted message privately.
